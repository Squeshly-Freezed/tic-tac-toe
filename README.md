What I've learned:

- First exposure to using async and await.
- Writing and learning super clean code through encapsulation (no spaghetti!) thanks to closures within factory functions and IIFE's.
- Practice and researched using 'strict'
- Array manipulation practice, especially with Array.some & Array.every & Array.map, and writing extensive array manipulation methods for the AI opponent.
- Trying to implement a Model View Controller architecture where there is a separation of concerns especially between the model and the view.
- Generally practicing previously learnt concepts including, CSS styling (especially keyframe animations, SVG content), JavaScript functions and syntax. HTML forms.

My thoughts on the project: I am blown away by how clean and organised my code is. It's a stark contrast to the spaghetti code that my calculator project was riddled with earlier in the course. It's absolutely beautiful to see and a breeze to work on clean code. I was slightly reliant on Claude here to fix a couple of bugs. I'm really proud to have implemented an 'intelligent' AI opponent with extensive board array manipulation. I also think the aesthetica overall are quite nice.
